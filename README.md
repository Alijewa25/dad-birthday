# 🎂 Dad's Birthday — 8 June 2026

A handcrafted tribute page I built for my father's birthday.  
No frameworks. No templates. Just HTML, CSS, and a lot of love.

---

##  About the Project

This is a personal gift — a single-page website I designed and built from scratch for my dad's birthday on **June 8, 2026**.

The page presents three life lessons I learned from him, written as personal reflections. The whole experience is meant to feel like opening a letter, not browsing a website.

> *"Sometimes the deepest love is expressed with the fewest words. But today, I wanted to say everything."*

---

##  Built With

- **Pure HTML & CSS** — zero dependencies, zero frameworks
- **Vanilla JS** — scroll-triggered animations, floating hearts
- **Google Fonts** — Playfair Display + Crimson Pro
- **CSS custom properties** — for a cohesive dark red & gold palette
- **IntersectionObserver API** — for smooth scroll reveals

---

##  Design Decisions

- **Color palette** — deep black (`#0a0505`) + crimson red + warm gold. Intentionally dramatic — because the person it's for deserves that.
- **Typography** — editorial serif fonts instead of the usual sans-serif defaults. Feels more like a handwritten letter.
- **No libraries** — keeping it lightweight so it loads instantly on any phone.
- **Mobile-first** — built with `clamp()`, `100svh`, and responsive grids so it looks right on the phone he'll open it on.
- **Grain overlay** — a subtle noise texture for depth without heaviness.

---

##  What I Learned

- How to use `IntersectionObserver` for scroll-triggered reveals without any JS library
- The difference between `100vh` and `100svh` (the latter handles mobile browser chrome correctly)
- How `clamp()` eliminates the need for most media queries in typography
- That the hardest part of a personal project isn't the code — it's finding the right words
